# Esercizio "Hello Java"

# Passaggi
1. Creare un progetto Java chiamato `hello-java`.
2. Aggiungere un package `org.lessons.java`.
3. Aggiungere una classe `StampaSaluto`.
4. Aggiungere il metodo `main` all’interno della classe appena creata.
5. Nel metodo main aggiungere l’espressione per **stampare in console un messaggio di saluto**.
